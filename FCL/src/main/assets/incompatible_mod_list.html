Incompatible Mods
此处列出常见的不兼容安卓的模组（包括不完全兼容/需要修改/主功能失效）。有关更多信息，请点击左下角详情按钮
Mods that are not compatible with Android.

Physics Mod
需要修补（将libPhysXJniBindings_64.so从Linux x86_64架构替换到安卓arm64/x86_64），否则启动后将会崩溃。
Needs patch (replace libPhysXJniBindings_64.so form Linux x86_64 to Android arm64/x86_64), or it will cause crash.
MCEF
无法在安卓上运行。（它通常被用来作为WebDisplays/In-game Browser浏览器）
Cannot work on Android. (It is usually used as a dependency of WebDisplays/In-game Browser.)
Valkyrien Skies/Valkyien Warfare
瓦尔基里天空/瓦尔基里战争，无法直接在单人模式下运行。你必须使用服务器运行（可以使用Termux），否则游戏会崩溃。
Cannot work directly in single player mode. You must use a server, otherwise it will crash the game.
Yes Steve Model
2.2.2+无法在安卓系统上运行，只能在x86_64的windwos/linux时运行
2.2.2+ cannot work on Android, only runs on x86_64(linux/windows)
IMBlocker
不适用于安卓。它会使游戏在启动阶段崩溃。
It is not for Android. It may make the game crash.
Replay
需要FFmpeg插件。你可以从谷歌商店下载或者下方链接下载
FFmpeg plugin is needed for Replay. You can download from Google Play Store or the following links.
https://github.com/FCL-Team/FoldCraftLauncher/releases/download/ffmpeg/Pojav.FFmpeg.Plugin.1.1.APK
https://pan.quark.cn/s/6201574edb62
Axiom
需要修补。使用未经修补的Axiom模组，启动后将会崩溃。 Needs patch, or it will cause crash.
Sodium/Embeddium
Sodium/Embeddium模组可能不适配GL4ES（OpenGL2）渲染器，可能会导致游戏崩溃。建议尝试支持OpenGL3/4的渲染器。
The Sodium/Embeddium mod may not support the GL4ES (OpenGL2) renderer, which could cause the game to crash. It is recommended to try a renderer that supports OpenGL3/4.
